# Environment requirements

1. Node, npm installed.
2. yarn installed `npm i -g yarn`
3. VScode extensions: **eslint**, **prettier**

# Install dependencies

```
yarn install
```

# Set up env variables

Copy file `.env.example` to `.env`

# Start developing server

```
yarn start:dev
```

# Deployment

```
yarn build
```

```
yarn start
```
